Cloud Storage

Cloud storage is a newly found and widely utilized feature that was created/ invented by a man called Joseph Carl Robnett Licklider in 1983. This is a service model that transmits and stores data upon remote storage systems. These systems are made avalable to there users over a network, such as the internet.
